The followings are the techniques we used in this website:

  • HTML:
  • Tables which have defined rows and cells
  • Insert gifs and photos that is resized by codes in the website, with alternate text
  • Basic HTML functions: title, headings, paragraphs, ordered and unordered list, attributes, hyperlink, line break and horizontal line, entities: space, adding comments
  • The <canvas> element is used to create a drawing area, and the <button> element is used to create clickable buttons on the web page
  • CSS:
  • Buttons which have two versions - hover and active
  • Basic CSS functions: changing fonts/ background colour by using span, id, and div, changing fonts (size and weight), adding comments
  • Max-width property is used to ensure that the canvas element does not exceed its container's width, and border property is used to add a border around the canvas
  • Javascript:
  • Pop-up window: alert()
  • Box for users to enter text: prompt()
  • Boolean expression/ logical operators: == (Equivalent), || (Or)
  • Creating Variables
  • Conditional statement: If-else
  • Let users draw on the canvas: canvas.getContext()
  • Add event listeners to the canvas and buttons: addEventListener
  • Get the mouse position: offsetX and offset
  • Draw lines on the canvas: beginPath(), moveTo(), lineTo(), and stroke()
  • Clear the canvas: clearRect()
  • Source of an image: toDataURL()
  • Create a temporary link: createElement()
  • Set the href and download attributes of the link element: setAttribute()
  • Click the link element to download the image: click()
