Background Information

In early 1995, Unisys began commercial charging based on the software patent of the LZW data compression algorithm it used in the GIF format. To avoid patent impact, the PNG used to represent a single image was created at the same time. IT expert Oliver Fromme named it as PING which was later shortened to PNG.


Main Features/ Functions

  1. Lossless compression
    PNG files are compressed using a derivative algorithm of the LZ77 algorithm, which uses a special encoding method to mark repeated data, so it has no effect on the colour of the image. This allows repeated saving without loss of image quality.
  2. Supports transparency effect
    PNG can define 256 transparency levels for the original image. Edges of the color image can be smoothly blended with any background and completely eliminating jagged edges. This feature is not available in GIF and JPEG.
  3. Indexed color mode
    The PNG-8 format uses an 8-bit palette to convert an RGB color image into an indexed color image. The representative color numbers selected from the image, each number corresponds to a color, and the amount of data in the image is therefore reduced. It is beneficial to the dissemination of colour images.

How is PGN better than the other image formats?

  1. As compared to GIF, PGN can handle millions of color options and can store images with more details.
  2. PNG is an open format with no patent, people can view and edit PNG files in a variety of programs without licenses.
  3. PNG images will not lose any data after compression, which makes storing and transferring files easier. This is a big advantage over lossy formats such as JPEG files, which degrades the image quality in the process of repeated compression and decoding.

How is PNG worse than the other image formats?

  1. Lossless compression makes the file size of PNG much larger than JPEG, but the file retains all the original data after compression, and the file takes up more storage.
  2. PNG can retain detailed and high-quality image data after compression, but at the same time, larger files will reduce the speed of web page reading and response. This means users have to make a trade-off between quality and speed.
  3. PNG was originally designed for the web, so the file does not support the CMYK colour model. Making conversion to print maybe difficult.
